ls(list) - выводит список содержимого в папке (к=вкл файлы и папки)
cd(change directory) - переход между папками
cd ../ - переход папками
ctrl +L - удалить строку в терминал
clear - удалить строку в терминал
pwd (path working directory)- абсолютный путь к папке в которой вы находитесь
ls -a - показать содержимое папки (включая скрытые папки и файлы)
rm -rf .(какая либо папка) - удаляет папку
git init инициализация git  репозитория
git remote add origin SSH - привязка локального репозитория к удаленному
git remote -v - посмотреть к какому репозиторию локально привязан
git remote remove origin - удалить привязку
